문제 정의 캔버스 사용 가이드

효과적인 문제 정의를 위한 단계별 안내

1. 문제 상황

• 관찰 가능하고 구체적인 상황을 작성하세요

• "학생들이 참여하지 않는다" → "발표 시간에 5명 이하만 손을 든다"

2. 영향받는 대상

• 직접적 영향: 문제의 당사자

• 간접적 영향: 학급 분위기, 학습 환경 등

3. 문제의 원인

• 5Why 기법 활용: "왜?"를 5번 반복하여 근본 원인 찾기

• 개인적 요인과 환경적 요인을 구분하여 분석

4. AIDT 도구 선정 기준

• 학생 수준에 적합한 도구 선택

• 교육적 목표와의 연계성 고려

• 접근성과 사용 편의성 확인

5. 성공 지표 설정

• SMART 기준: 구체적, 측정가능, 달성가능, 관련성, 시한성

• 정량적 지표와 정성적 지표의 균형
